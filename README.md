# Promise
write a promise polyfill.

## 工程概述
- 使用 [Rollup](https://rollupjs.org) 进行 umd 模块打包 
- 使用 [TypeScript](https://www.typescriptlang.org) 开发
- 使用 [Tslint](https://palantir.github.io/tslint/) 进行代码风格检查
